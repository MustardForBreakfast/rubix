# Rubix
A two-dimensional Javascript Rubik's Cube, complete with rudimentary solver. Feel free to check it out yourself here: https://polar-thicket-78330.herokuapp.com/

## Getting Started

    $ npm install
    $ npm start

## Stack

 - Node.js
 - Express.js
 - JQuery

## Plans for the Future
  - Improved UX
